# esphome-build
ESPHome Build Environment Container

## Usage
In your Containerfile
```
ARG ESPHOME_BUILD_VERSION=latest
FROM ghcr.io/bbusse/esphome-build:${ESPHOME_BUILD_VERSION}
```
